# CLM Setup — Super App Flow

Interactive prototype for the full Rapsodo CLM PRO PC onboarding flow — Account Setup, Room Confirmation, Installation Verification, and Connect & Calibrate.

Single-file HTML/CSS/JS, no build step.

## Run

Open `index.html` directly in a browser, or serve locally:

```bash
npx serve .
```
